## Description

A minimal Unix shell written in C that implements tokenization, parsing, variable
expansion, heredocs, redirections, pipelines, builtins (`echo`, `cd`, `pwd`,
`env`, `export`, `unset`, `exit`), and execution of external programs.

## Instructions

From the repository root:

```bash
make
./minishell
```
